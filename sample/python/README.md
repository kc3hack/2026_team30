# Pythonの検証用フォルダ

## 環境構築

1. フォルダを移動する `cd sample/python`
2. 仮想環境の作成(初回のみ) `python -m venv venv`
3. venvの起動 `.\venv\Scripts\activate`
4. ライブラリのインストール(更新時のみ) `pip install -r requirements.txt`
