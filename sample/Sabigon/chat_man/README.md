# chat_man

ブラウザで動作する簡易チャット + 議事録入力のフロントエンドです。Vite + React + TypeScript で構成しています。

## できること
- テキストチャットの送信（色・文字サイズを変更可能）
- マイク録音（ブラウザの音声入力で録音して再生）
- 議事録／文字起こし用のメモ入力

## 画面とルーティング
- `/` : ホーム（チャット／議事録への導線）
- `/chat` : チャット画面
- `/minutes` : 議事録入力画面

## セットアップ
```bash
npm install
```

## 開発サーバー起動
```bash
npm run dev
```

## ビルド
```bash
npm run build
```

## 注意点
- 録音はブラウザのマイク権限が必要です。
- 送信内容はブラウザ内のみで保持され、リロードで消えます（バックエンド未接続）。

## 技術構成
- React 19
- TypeScript
- Vite
- React Router
